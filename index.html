Técnologias que utilizamos

Estas son solo algunas de las técnologias que usamos en el dia a dia para crear sitios y experiencias web.

HTML5

Ultima versión del HTML para estructurar sitios web permitiendo a los sitios Web y a las aplicaciones ser más diversas y de gran alcance

CSS3

Última versión del lenguaje que define la apariencia de las páginas web

JAVASCRIPT

Lenguaje de programación para crear interactividad dinámica en los sitios web

BOOTSTRAP

Framework para crear interfaces de usuario limpias y totalmente adaptables a todo tipo de dispositivos y pantallas.

ANGULAR.JS

Framework para el desarrollo de aplicaciones de lado cliente con Javascript..

JQUERY

Librería de Javascript que nos permite hacer páginas web dinámicas

WORDPRESS

Sistema de gestión de contenidos (CMS) mas popular que permite crear y mantener un blog o un sitio web..

PRESTASHOP

Plataforma de comercio online de código abierto líder en el mercado del comercio electrónico.

Técnologias que utilizamos

  • Diseño Web a Medida

    Diseñamos páginas web de acuerdo a tus necesidades y las de tus clientes, enfocándonos en en tu marca haciendolas escalables para ser competitivos.

  • Diseño UX

    Tomando en cuenta las necesidades de los usuarios, diseñamos la interfaz, contenido y las intereacciones del sitio web.

  • Diseño de Interacción

    Definimos las formas de operar la interfaz mediante teclado, mouse, touch, o una combinación de ellos, los flujos de operación y las respuestas del sitio web.

  • Mercado Objetivo

    Enfocamos el diseño e intercaccion de los sitios web a al grupo de consumidores ideales de tu producto o servicio.
